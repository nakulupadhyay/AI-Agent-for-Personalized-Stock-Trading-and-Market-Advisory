## ⚡ What is this?

Welcome to the brains of the operation. 

The `ml-service` is an independent, state-of-the-art Python microservice built on FastAPI. It handles all the heavy lifting for the **AI Stock Trading and Market Advisory Platform**. 

By decoupling our data science stack from our Node.js transactional backend, we ensure that deep learning inference (which is highly CPU/GPU bound) never blocks our web server. This service ingests market data, runs it through our proprietary ensemble models, and returns actionable JSON insights—Buy/Sell/Hold signals, sentiment scores, and risk classifications—in milliseconds.

---

## 🛠️ The Tech Stack

Built for data processing, model serving, and low latency:

- **FastAPI:** The fastest, most modern Python web framework. Period. Perfect for creating robust microservices with auto-generated documentation.
- **TensorFlow / Keras:** Used for sequence modeling. Our LSTM (Long Short-Term Memory) networks analyze years of historical price data to predict short-term trends.
- **PyTorch & Transformers (Hugging Face):** We utilize a fine-tuned `FinBERT` model to run natural language processing (NLP) on real-time financial news headlines to gauge market sentiment.
- **XGBoost & Scikit-Learn:** Employed for classification tasks, specifically generating trade recommendations based on technical indicators and user risk profiling.
- **Pandas, NumPy, & TA:** The core data manipulation toolkit. We use the `ta` library to compute RSI, MACD, Bollinger Bands, and more on the fly.

## 🚀 Getting Started

Let's boot up the inference engine.

### Prerequisites
- Python (v3.9+)
- pip
- Virtual Environment (`venv`)

### Installation

1. **Create and Activate a Virtual Environment:**
   ```bash
   python -m venv venv
   
   # Windows:
   venv\Scripts\activate
   
   # macOS/Linux:
   source venv/bin/activate
   ```

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the `ml-service/` root directory:
   ```env
   MONGO_URI=mongodb://localhost:27017/ai-stock-trading
   MODEL_DIR=./models
   ```

4. **Start the FastAPI Server:**
   ```bash
   uvicorn app.main:app --reload --port 8000
   ```
   *The API will be available at `http://localhost:8000`.*
   *View the auto-generated Swagger documentation at `http://localhost:8000/docs`.*

## 🧠 The Models

Our recommendations are generated by pipelining three distinct ML approaches:

1. **Technical Foundation (LSTM):** Identifies support/resistance and momentum based strictly on historical price/volume action.
2. **Sentiment Overlay (FinBERT):** Adjusts the technical outlook based on how the media and internet are currently talking about the asset. Good charts don't matter if the CEO just resigned.
3. **Risk Constraint (Random Forest):** Filters the final recommendation through the user's specific risk profile. High-volatility assets are blocked from conservative portfolios.
